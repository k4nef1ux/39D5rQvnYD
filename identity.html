the identity kit

a store nobody trusts converts nothing, and trust is mostly consistency: same name, same face, same line, everywhere. run these in order - each output feeds the next. never touch it again until something sells.

1 · the brand name

rules before the prompt: short (under 9 letters), easy to say out loud, nothing descriptive - descriptive names die when you pivot niches. check that the .com or a clean .co is available.

pick by the first-not-best rule: the first one that is available and you can say out loud without wincing.

2 · the handle ladder

check the name on every platform before committing. when the clean handle is taken, fall down this ladder in order - never sideways into a different name:

  1. [name] - the clean handle
  2. [name]shop or [name]store
  3. [name]hq
  4. get[name] or shop[name]
  5. [name]official

whichever rung you land on, use the same rung on all seven platforms. consistency beats cleanliness.
