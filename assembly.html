the assembly

this is the machine. the drops are just pulls of the handle. block one afternoon, go top to bottom, tick every box. after today you never do this again.

1 · the store (45 min)

  • create the shopify store - starter plan is enough
  • connect payments (shopify payments or paypal) and run the test-mode checkout
  • set currency + shipping zones (worldwide unless you have a reason not to)
  • delete the sample products. pick the plainest free theme, dark if available
  • set the store name + logo from identity

done when: the storefront loads at your url with zero placeholder content.

2 · fulfillment (30 min)

  • create the printify account, connect it to the store
  • lock your defaults per module (the sticker module locks kiss-cut stickers - see module 01)
  • order nothing. printify only charges when a customer buys

done when: a test product pushes from printify to the storefront without you touching anything twice.

3 · the seven handles (45 min)

same handle everywhere. generate it in identity before you register anything.

  • paste the matching bio from identity into each
  • same avatar everywhere. store link in every bio slot

done when: searching your handle on any platform finds the same face.

4 · templates, saved once (45 min)

  • product page template: title / 3 bullets / 40-word description / shipping faq - saved as a draft product you duplicate
  • pin template: 1000 x 1500 in canva - product image + 4-word hook + handle
  • shorts structure (memorize it): hook 0-2s, product or design reveal 2-15s, cta last 3s
  • threads / x structure: hook line, one image, one cta line
  • bookmark the module you are running so its prompts are one click away

5 · the log (15 min)

build the sheet from operate · the drop log. one row per drop. no drop exists until it has a row.

assembly is done when

  • a $1 test purchase completes end to end
  • all seven socials can post and carry the same bio + avatar + link
  • the product page template duplicates in under 30 seconds
  • the log sheet has its header row and drop #1 waiting empty

now open module 01. tonight.
