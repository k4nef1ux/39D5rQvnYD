# 5 minute offers

a static product site. no build step, no dependencies - open `index.html`
or serve the folder with any static host (github pages, hostinger, netlify).

- `index.html` - home: the manifesto, the system, the operating rules
- `assembly.html` - the one-time setup day
- `identity.html` - name, handles, bios, about pages (prompt-generated)
- `vault.html` - the master prompt library
- `operate.html` - pricing, the 48-hour kill-or-scale rule, the drop log,
  the receipts loop, friction fixes
- `modules/stickers.html` - module 01: stickers on printify + shopify,
  end to end, a prompt for every field
- `assets/style.css` + `assets/site.js` - the whole design system and the
  only script (copy buttons + active nav)

note: this repo previously held the findshq gift-site scaffold. it remains
recoverable in git history at commit bc5b6c0.
